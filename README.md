# BankSystem 🏦

Basic Bank System in C++ to manage clients and accounts.  

---

## 📝 Description
This is a simple bank system without transaction functionality.  
- Manage clients and accounts  
- View account balances  
- Add or remove clients  

---

## 🎮 How to Use
1. Open the project in Visual Studio or any C++ compiler.  
2. Run the program.  
3. Follow the menu to add, view, or remove clients.  

---

## 🌟 Features
- Client management  
- Account creation and deletion  
- View account details
